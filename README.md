# 餐廳清單 

此專案僅提供搜尋關鍵字功能


### 安裝

1.開啟終端機(Terminal)cd 到存放專案本機位置並執行:

```
git clone https://github.com/Chris-Chou518/Restaurant_List.git
```

2.初始

```
cd Restaurant_List_  //切至專案資料夾
```

```
npm install  //安裝套件
```

3.安裝nodemon套件

```
npm i nodemon  
```


4.開啟程式

```
nodemon app.js  //執行程式
```

請至[http://localhost:3000](http://localhost:3000)開始使用程式

# 功能
-搜尋關鍵字
