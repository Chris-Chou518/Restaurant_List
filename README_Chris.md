# Restaurant list

It is a list about different restaurants.

## Getting Started

1. Clone this project by terminal
```
https://github.com/Chris-Chou518/Restaurant_List.git
```

2. Install
```
cd Restaurant_List
```
```
npm install 
```
```
npm i nodemon
```

3. Find app.js and execute it
```
cd  app.js
```
```
nodemon app.js
```
4. Use this project at http://localhost:3000
